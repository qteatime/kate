Packaging cartridges
====================

Each Kate cartridge contains a special section that describes how the
console (and catalog) should present the cartridge to the player, and
how to turn all of the file in the cartridge into a game it can run.

For a quick introduction to packaging cartridges, you should check
out the :doc:`developer setup guide </dev/manual/intro/setup>` and the
:doc:`creating your first cartridge tutorial </dev/manual/intro/hello-world>`.

Kate cartridges and how you use that in yours:
