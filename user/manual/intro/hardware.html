Tech specs#

The console#

Kate is a console that uses 2020s technology to recreate some of the feelings of early 2000s and 2010s technology and culture. It sports a cute strawberry cake × cats themed case, specific security feedback channels, and a proper cartridge slot.

The hardware comes in two versions. The initial hand-held (Pancake) version:

And the shiny new mini-Kate box (Cheesecake) version, which you can certainly hold in one hand but not really see much without plugging it to some external display through HDMI:

Note

The hardware, 3d models for the case design, operating system, and all other components are planned to be released as open-source in 2024, so you will be able to build your own physical Kate console from these parts.

Right now only the Kate Emulator is available as an early technology preview. The physical console will run the same software specification and thus be compatible with all Kate cartridges that run in the emulator, but with the hardware limitations specified below, unless you build your console using custom hardware.

Note

The cases and operating system are designed to use off-the-shelf Raspberry Pi boards. You should be able to use other SoCs, but you’ll need to modify at least the case and PCB designs, and you might need to patch Raspberry OS-specific components if you’re using another Linux distribution.

Graphics#

Kate uses 4” capacitive multi-touch (10 points) display with a resolution of 800x480. When connecting to an external display, it can reach the maximum resolution of 1280x720. The screen aspect ratio is 5:3.

The GPU supports OpenGL ES 3.1.

Note

When running the emulator, this means that your GPU card should have at least been released after 2012, and have updated drivers that provide the needed functionality.

Input#

Kate has a total of 12 digital input buttons used for controlling games:

  • (Ok), (Sparkle), (Menu), and (Cancel) face buttons;

  • (D-Pad);

  • (Capture button) and (Berry button);

  • and shoulder buttons;

It also has a capacitive touch screen with support to 10 touch points, a haptics module for vibration, a 3-axis gyroscope sensor, and a 3-axis accelerometer sensor.

Up to 4 gamepads can be connected to the console at a time, either through bluetooth or directly through USB.

Cartridges#

Games for Kate can be made with many different technologies, and are shared as ROM files, which Kate calls a “cartridge”. The ROM files incldue all the data needed to run the game. There are no limits for the ROM file size.

Kate never runs the ROM files directly; rather they’re copied into the console in a more efficient format upon installation. From that point on, as long as you don’t archive the game in Kate, it can be played without needing to keep the file around or insert the disc [1].

[1]

Kate cartridges contain no form of copy-protection, and rather rely on players being respectful of game developers’ wishes regarding sharing and modifying the game—such wishes are included in the cartridge’s metadata.
